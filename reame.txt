Кроссплатформенный файловый менеджер для работы с Android-устройствами на различных ОС
Был написан, потому что:
1. По роду своей деятельности приходится работать с мобильными устройствами из под различных ОС (особенно грустно с этим в macOS);
2. Это интересно.

Умеет в:
1. Навигацию по файловой системе (в том числе ходить по симлинкам);
2. Передавать файлы из/на ББ;
3. Редактировать файлы;
4. Смотреть список установленных приложений и удалять ненужные;
5. Делать скриншоты.

Прежде чем начать использовать софт и/или исходный код, убедись, что:
1. У тебя установлен JDK;
2. ADB видит подключенное к ББ устройство (для проверки введи в консоли "adb devices")

Если устройство рутовано, то можно установить в файле settings.ini параметр startAsRoot в значение true
